CARA PAKAI:
1. Upload isi folder ini langsung ke root repository GitHub Pages.
2. Pastikan index.html berada di halaman utama repository, bukan di dalam folder lain.
3. Buka Settings > Pages > Deploy from branch > main > /(root).

FITUR:
- Cover pembuka gerbang Jawa
- Efek daun jatuh
- Efek kabut
- Musik
- Countdown acara
- Detail akad dan resepsi
- Google Maps
- Love Story
- Galeri foto
- RSVP dan ucapan tersimpan di browser
- Amplop digital dan copy rekening
- Bottom navigation
- Responsive HP/PC

CARA EDIT DATA:
- Nama pasangan: edit di index.html.
- Tanggal countdown: edit file assets/js/script.js pada bagian:
  const weddingDate = new Date('2026-09-21T08:00:00').getTime();
- Alamat dan jam acara: edit bagian Detail Acara di index.html.
- Rekening: edit bagian Amplop Digital di index.html.
- Foto:
  Ganti file SVG di assets/img/ dengan JPG/PNG, lalu sesuaikan nama file di style.css.
- Musik:
  Ganti assets/audio/music.mp3 dengan file MP3 asli.
