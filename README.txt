UNDANGAN DIGITAL JAWA VINTAGE
Muhammad Hibatulloh & Maulidin Nimaturrosyidah

Cara menggunakan:
1. Ekstrak folder undangan_jawa_vintage.zip.
2. Buka file index.html di browser.
3. Edit data acara di index.html:
   - Nama orang tua
   - Tanggal akad/resepsi
   - Alamat acara
   - Link Google Maps
4. Edit tanggal countdown di assets/js/script.js pada bagian:
   const weddingDate = new Date('2026-12-28T08:00:00').getTime();
5. Untuk musik, masukkan file MP3 ke:
   assets/audio/music.mp3
6. Untuk foto, bisa mengganti kotak galeri dan foto mempelai dengan tag <img>.

File utama:
- index.html
- assets/css/style.css
- assets/js/script.js

Catatan:
Template ini menggunakan Bootstrap, Google Font, dan AOS Animation via CDN, sehingga internet dibutuhkan agar font dan animasi tampil sempurna.
